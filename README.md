# Dota 2 战绩管理系统

## 项目背景

这个项目诞生于与 Dota 2 好友们的日常对黑时光。当我们五个人组队与其他好友进行 5v5 内部对战时，发现需要一个方便记录对黑战绩的平台。

最初，我们只是简单地用 Excel 或纸质笔记本来记录比赛结果，但这样很难进行统计分析，也无法直观地看到每位队员的表现。随着对黑场次越来越多，我们决定开发这个系统来：

- 🏆 记录每场对黑的结果和详细数据
- 📊 统计每位玩家的胜负率、擅长英雄、常用位置
- 🎮 追踪团队的成长轨迹和历史战绩
- 🔍 分析不同英雄组合的胜率

当前版本为 **v0.1 初版**，已实现基础的 CRUD 功能和数据可视化。后续计划接入 OpenDota API 来自动化获取真实战绩数据。

## 当前功能

### 核心功能
- ✅ 玩家管理：添加、编辑、删除玩家信息
- ✅ 英雄管理：导入、查看英雄数据
- ✅ 比赛记录：创建、查看、管理对黑比赛
- ✅ 数据统计：胜负率、英雄使用统计、玩家排名
- ✅ 数据可视化：图表展示历史趋势和统计数据
- ✅ MVP 标注：标记每场比赛的最有价值玩家

### 技术栈
- **前端**: React 19, Vite, TailwindCSS 4.x, Radix UI, Wouter
- **后端**: tRPC v11, Drizzle ORM, Express
- **数据库**: MySQL
- **包管理**: pnpm

## 快速开始

### 安装依赖
```bash
pnpm install
```

### 开发环境
```bash
# 启动开发服务器
pnpm dev
```

服务器将自动检测可用端口（3000-3020），前端和后端会自动开启热重载。

### 构建生产版本
```bash
pnpm build
pnpm start
```
